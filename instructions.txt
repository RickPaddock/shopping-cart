In VSCode bash terminal to pull/download image to docker desktop: 
docker pull hseeberger/scala-sbt:8u222_1.3.5_2.13.1


In VSCode bash terminal to run container. Opens interactive terminal:
docker run -it --rm hseeberger/scala-sbt:8u222_1.3.5_2.13.1 bash

# To run image as container and bind C drive to internal folder in docker
docker run -it --name scala-shopping-cart --mount type=bind,source="C:/Users/rickp/Documents/projects/Scala/shopping-basket",target="/home/shopping-basket" hseeberger/scala-sbt:8u222_1.3.5_2.13.1

# I then created the folder structure, inside the terminal (--name=. uses current dir as project name): 
-- in /home/shopping-basket folder within docker --
sbt new scala/scala-seed.g8 --name=.


# TO STOP
docker stop scala-shopping-cart

# TO START
docker start -i scala-shopping-cart
